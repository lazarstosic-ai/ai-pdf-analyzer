# AI PDF Analyzer

AI system for analyzing academic PDF documents using LLMs and RAG architecture.

## Features
- PDF upload
- AI summarization
- Keyword extraction
- DOI detection
- IMRAD analysis

## Technologies
- Python
- FastAPI
- LangChain
- OpenAI API
- ChromaDB

## Installation
pip install -r requirements.txt

## Run
uvicorn main:app --reload
